# EventMate AI - Event Management Platform

## 🚀 How to Run

### Backend (Spring Boot)
```bash
cd eventmate-backend
./mvnw spring-boot:run
```
Backend runs at: http://localhost:8080

### Frontend (React)
```bash
cd eventmate-frontend
npm install
npm start
```
Frontend runs at: http://localhost:3000

---

## 🗂 Project Structure
```
Event-Mate-organizer/
├── eventmate-backend/     → Spring Boot (Java)
└── eventmate-frontend/    → React.js
```

## 👤 Roles
- **User** → Browse events, book seats, make payments, give feedback
- **Organizer** → Add/manage events, monitor bookings, send reminders, view reviews
- **Admin** → View all users, organizers, events, bookings
